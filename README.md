# nixos-install
Install-Script for NixOS. I use it mostly to quickly deploy new servers within seconds.

### Usage
```
bash <(curl -sSf https://raw.githubusercontent.com/fxttr/nixos-install/master/install-zfs.sh) <disk> <hostname>
```
